## Folk Toy Winter Tale

This single-page Next.js experience tells the story of handmade folk toys that awaken to celebrate the New Year. A looping cinematic timeline guides the viewer from a frosted winter window into a warm holiday room where clay, fabric, and wooden toys come to life.

### Local development

```bash
npm install
npm run dev
```

The app is available at [http://localhost:3000](http://localhost:3000). Edit `src/app/page.tsx` to adjust the narrative, visuals, or timing. Styles and handcrafted character details live in `src/app/globals.css`.

### Production build

```bash
npm run build
npm run start
```

### Deployment

The project is optimized for Vercel. Run `vercel deploy --prod` with the proper project name and token to publish updates.
